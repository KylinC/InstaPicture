## 简介

SJTU-EE458 Course Project, Spring 2020.

一款用react和python开发的图片社交软件，支持内容推荐、好友圈、用户相册等功能。



## 技术栈

- react
- react-redux
- react-router4

- express
- nodejs
- mongoose
- MongoDB
