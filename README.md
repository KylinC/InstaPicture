# InstaPicture 1.1

> SJTU-EE458 Course Project, Spring 2020.

> InstaPicture is a photo-sharing application where users can upload photos/smallposts/videos to share with followers

> Add content page and personal page



## Technology stack

### Front-End

- react
- react-redux
- react-router4
- es6

### Back-End

- express
- nodejs
- mongoose
- MongoDB
